Compilio is an online compiler you can use to compile your files. It currently supports LaTeX files. You can choose either to compile them using the drag & drop section or download a lightweight python client to do it remotely.

$ # Prefix your command with compilio
$ compilio pdflatex my_file.tex
$ # Your file is compiled on the compilio server
$ # and the result is downloaded
