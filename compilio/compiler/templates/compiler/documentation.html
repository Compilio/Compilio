User's guide

Compilio can be used in two different ways. You can either upload the files you want to compile directly on the homepage or use Compilio CLI, a command-line solution to compile your files directly from your shell.

Using Compil.io

  1. Go to Compilio's homepage and drop the files you want to compile.
  2. You can choose from the different compilers available by clicking on one of the tasks displayed on the homepage.
  3. Wait until the compilation finishes. You can then see the compilation trace code and download the result files.
  4. If you want to save the task and make it private, create an account and go back to the task page once connected.

Using Compilio CLI

  1. Download and install Compilio CLI using PIP.
  2. Run the your compilation command prefixed with compilio. For example, you can run pdflatex by running compilio -- pdflatex file.tex
  3. Wait until the compilation finishes. The compilation trace is displayed on your shell and the files are directly downloaded, as if the compilation task ran locally!

Host's guide

You can deploy your own copy of Compil.io as a regular Django application. You can also use your own server as a task runner, managed automatically by the main Compil.io instance.

To see how to install your own Compilio instance, see the contribution file.

Contributor's guide

You can contribute to Compil.io directly on GitHub!
